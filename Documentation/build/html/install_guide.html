Installation Guide

  1. Database

    You need to install the postgres and pgAdmin (https://www.pgadmin.org/ ) for using postgres sql. For full guide on how to install please refer this link.
    Some important points:

    • If you have a database in sql and want to change it to psql use this git link .
    • If you have a csv and want to import data to pgAdmin database use psql shell with this command
      \copy Public.”table _name” FROM ‘csv_file_location’ DELIMITER ‘,’ CSV HEADER null ‘null’;
      Example:
      \copy Public.”elsi_college_dtls” FROM ‘C:\example\data.csv’ DELIMITER ‘,’ CSV HEADER null ‘null’;

      Sometimes you need to provide encoding details also of the csv file, so just provide extra parameter encoding ‘UTF8’ where encoding may have any other format other than UTF8 depending on the excel version and file type you are using (.csv, .xlsx).

    • If you have .pgsql file and want to import its data into pgAdmin then open command prompt with administrator rights, also make sure that your environment variables path are set for that particular location if not, then set the environment variables path. Type the following command
      psql –U username dbname < dbexport.pgsql
      where username is the username of pgadmin and dbname is the database name that you want to import data into .
      Example:
      psql –U postgres elsi < C:\example\data.pgsql

    Note: Make a database named elsi in pgadmin. A folder named test_database is provided which contains data.pgsql file this is the testing file which can be used for testing the app (for importing this file follow step c), later on just truncate the tables and import your own data.
    For setting the environment variables if not set already :
    for windows 10

    1. In Search, search for and then select: System (Control Panel)
    2. Click the Advanced system settings link.
    3. Click Environment Variables. In the section System Variables, find the PATH environment variable and select it. Click Edit. If the PATH environment variable does not exist, click New.
    4. 4. In the Edit System Variable (or New System Variable) window, specify the value of the PATH environment variable. Click OK. Close all remaining windows by clicking OK.( the path value which you need to write here will be C:\Program Files\PostgreSQL\12\lib and C:\Program Files\PostgreSQL\12\bin these locations may change depending on where you have installed the PostreSOL in system).
    5. Reopen Command prompt window, and then run the respective command.

  2. Back end and Front end

    Create a new project folder and clone the git from here . For some big folders which are not present in git repo. consider downloading them from this emails 5a.eysip2020@gmail.com Google Drive and save these folders in the vue_frontend/app folder.

    Backend :
    The django_backend folder contains the backend system. Make sure that you have latest package of django and python installed. Install all the dependencies using the command in django_backend/app folder:
    pip install -r requirements.txt
    Open the command prompt in django_backend/app folder and follow these steps:

    1. python manage.py makemigrations <appname>
      -->python manage.py makemigrations main
    2. python manage.py migrate
    3. for creating superuser for django admin: python manage.py createsuperuser
    4. then finally python manage.py runserver <port>
      -->python manage.py runserver 8081
    The backend sever will start at localhost 8081.
    To visit the django admin panel type in browser http://127.0.0.1:8081/admin and enter the superuser credentials.
    Note: If you want to change localhost server on different port just change 8081 with your preferred port and provide the same port in settings tab present in GUI of app .

    Frontend :
    The vue_frontend folder contains the frontend system. Make sure that you have latest package of npm installed. Install all the dependencies.
    Open the command prompt in vue_frontend/app folder and follow these steps:

    1. npm run electron:serve
    The app will start on localhost.
    Note: If you want to run the app in production environment :
    Open the command prompt in vue_frontend/app folder and follow these steps:
    1. npm run electron: build
    We will get an setup file and executable file with some other files in dist_electron and dist_electron/win-unpacked folder.
